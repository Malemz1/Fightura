## What it does

<CardGrid>
    <Card title="Body follows EF pose" icon="random">
        Figura body parts that match a known Epic Fight joint anchor to it automatically. Roll, dash, attack — the avatar tracks the real combat pose.
    </Card>

## Quick install

1. Download `fightura-1.0.0.jar`
2. Drop it into your client `mods/` folder alongside Figura `0.1.5+1.20.1-forge` and Epic Fight `20.14.15.1+`
3. Launch Minecraft

That's the whole setup. See the [Installation guide](/install/) for the long version.

## How it works in one paragraph

Fightura attaches a render layer to Epic Fight's player renderer. Each frame it captures the current Epic Fight pose, then for every Figura part that maps to a known Epic Fight joint (by `parentType`, by part name, or by Lua override), the part's transform is set to follow that joint. Parts without a mapping fall through to Figura's normal render path and inherit through their parent. One pipeline handles accessory avatars, full-body avatars, and bend-bone rigs the same way.
