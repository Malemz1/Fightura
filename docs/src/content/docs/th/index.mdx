## ติดตั้งเร็ว ๆ

1. โหลด `fightura-1.0.0.jar`
2. โยนเข้าโฟลเดอร์ `mods/` ของ client พร้อม Figura `0.1.5+1.20.1-forge` และ Epic Fight `20.14.15.1+`
3. เปิดเกม

จบเลย ดูแบบละเอียดที่ [คู่มือติดตั้ง](/Fightura/th/install/)

## ทำงานยังไง (สรุปย่อ)

Fightura ติดตั้ง render layer เข้ากับ player renderer ของ Epic Fight ทุก frame มันจะเก็บท่าปัจจุบันของ Epic Fight แล้วสำหรับทุก ๆ Figura part ที่ map ไปยัง joint ที่รู้จัก (ผ่าน `parentType`, ชื่อ part, หรือ Lua override) — part นั้นจะถูก set transform ให้ตาม joint ส่วน part ที่ไม่มี mapping จะ fallback ไปยัง render path ปกติของ Figura และ inherit จาก parent โดยอัตโนมัติ pipeline เดียวจัดการ avatar accessory, full-body, และ bend bone rig ทั้งหมดเหมือนกัน
